Definición de FrontEnd

Son aquellas tecnologías que corren el lado del cliente, concretamente en el navegador, cuando hablamos de desarrollo web, existiendo tres lenguajes involucrados en este aspecto, como son HTML, CSS y JAVASCRIPT. Un desarrollador especializado en FrontEnd, trabaja de una manera u otra, estas tres tecnologías. Vinculadas a ellas existen otras que trabajan la presentación como Bootstrap, JQuery, otras para la transferencia de información, como xml, json y diferentes Frameworks que potencian el desarrollo en Javascript, como por ejemplo Angular.

Definición de BackEnd

No menos importante es el BackEnd, que es la parte que se encuentra en el lado servidor, hablamos de los Servidores de Aplicación y de Bases de Datos, cuyas desarrollos pueden estar implementados en lenguajes como PHP, Java, Net etc, y sus respectivos framework y se encargan de interactuar con la Base de Datos, como por ejemplo PostgrepSql, MySql, Oracle o SqlSever entre las más comunes, sin menospreciar las nuevas tendencias en Bases de datos NoSQL.
